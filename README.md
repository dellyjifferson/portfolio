##  Running Locally

To view this portfolio on your local machine:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/dellyjifferson/portfolio.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd portfolio
    ```
3.  **Open `index.html` in your browser:**
    You can simply double-click the `index.html` file, or use a live server extension in VSCode for a better experience.
